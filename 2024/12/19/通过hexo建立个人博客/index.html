通过hexo建立个人博客

通过 Hexo 建立个人博客及多台电脑维护的完整流程总结：
一、搭建 Hexo 博客

  1. 环境准备
    安装 Node.js 和 npm（Node.js 自带）。
    安装 Git：用于版本控制和部署。
  2. 初始化 Hexo
    bash
    复制代码
    npm install -g hexo-cli # 安装 Hexo CLI
    hexo init myblog # 初始化博客项目
    cd myblog
    npm install # 安装依赖
  3. 生成博客
    运行以下命令，预览博客：

bash
复制代码
hexo generate # 生成静态文件
hexo server # 启动本地服务器，访问 http://localhost:4000
二、部署到 GitHub Pages

  1. 创建 GitHub 仓库
    创建一个名为 yourusername.github.io 的仓库，用于部署博客。
    （可选）再创建一个仓库如 myblog，用于存储 Hexo 源文件。
  2. 安装部署插件
    在 Hexo 项目目录下安装 hexo-deployer-git 插件：

bash
复制代码
npm install hexo-deployer-git –save
3. 配置 _config.yml
在项目根目录的 _config.yml 文件中，找到 deploy 部分，配置如下：

yaml
复制代码
deploy:
type: git
repo: https://github.com/yourusername/yourusername.github.io.git
branch: main # 或者 master，根据你的仓库设置
4. 部署博客
运行以下命令：

bash
复制代码
hexo clean # 清理旧文件
hexo generate # 生成静态文件
hexo deploy # 部署到 GitHub Pages
你的博客将可通过 https://yourusername.github.io 访问。

三、更新博客文章

  1. 创建新文章
    运行以下命令：

bash
复制代码
hexo new “My New Post”
这将在 source/_posts/ 目录下生成一个 Markdown 文件。

  1. 编辑文章
    打开生成的 Markdown 文件，编写内容，并支持图文混排。常用语法：

markdown
复制代码

标题

副标题

加粗文字
斜体文字
代码块

链接描述
3. 更新博客
每次更新文章后，执行以下命令：

bash
复制代码
hexo clean
hexo generate
hexo deploy
四、多台电脑维护 Hexo 博客

  1. 将 Hexo 项目存储在 Git 仓库
    创建一个新仓库（如 myblog）用于存储 Hexo 项目：

bash
复制代码
git init
git remote add origin https://github.com/yourusername/myblog.git
git add .
git commit -m “Initial commit”
git branch -M main
git push -u origin main
在 .gitignore 文件中，确保忽略以下内容：

plaintext
复制代码
node_modules/
public/
.deploy_git/
db.json
2. 在另一台电脑克隆项目
在新电脑上，克隆 Hexo 项目并安装依赖：

bash
复制代码
git clone https://github.com/yourusername/myblog.git
cd myblog
npm install
3. 同步更新
在任何一台电脑上更新 Hexo 博客时：

将本地更改推送到 Hexo 项目仓库：

bash
复制代码
git add .
git commit -m “Update post”
git push
在另一台电脑上，拉取最新代码：

bash
复制代码
git pull
部署博客到 GitHub Pages：

bash
复制代码
hexo clean
hexo generate
hexo deploy
五、优化建议
图片管理：

使用 Hexo 插件 hexo-asset-image，将图片与文章文件绑定管理。
或者上传图片到图床（如 GitHub 图床、阿里云 OSS）。
主题美化：

替换 Hexo 默认主题，推荐 NexT 或其他 Hexo 主题。
按需修改主题的 _config.yml 文件，调整外观和功能。
搜索功能：

安装 hexo-generator-searchdb，添加站内搜索支持。
备份：

Hexo 源文件和博客部署分别存在两个仓库，确保代码和网站安全。
